# Auto-haul (give-haul leaf) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Idle pawns autonomously carry loose ground items into a stockpile zone — closing Layer 4's autonomous work loop.

**Architecture:** Add one pure JobGiver, `give-haul`, to the data think-tree (`sim.think`). It selects the nearest reservable loose item (grounded, not already in a stockpile) and mints the existing `:haul` job toward the nearest stockpile cell. It slots between the eat (survival) and wander (idle) leaves. No changes to the job FSM, reservations, or the walker — `give-haul` self-guards by returning `nil`, exactly like `give-eat`/`give-wander`.

**Tech Stack:** Clojure 1.12, cognitect test-runner (`clj -M:test`). Headless tests (no libGDX).

**Spec:** `docs/superpowers/specs/2026-05-27-auto-haul-design.md`

- [ ] **Step 4: Add the `give-haul` giver in `src/sim/think.clj`**

`manhattan` is already defined (`defn-`, used by `give-eat`). Insert `give-haul` immediately AFTER `give-wander` and BEFORE the `givers` map:

```clojure
(defn- give-haul
  "Among grounded items NOT already in a stockpile that this pawn can reserve,
   pick the nearest to the pawn (Manhattan, ties by :id) and haul it to the
   nearest stockpile cell to that item (ties by [x y]). Nil when there is nothing
   to haul or no stockpile exists, so the tree falls through to wander. An item
   already on a stockpile cell is excluded, so a hauled item is never re-picked
   (self-terminating). In RimWorld terms this is the :hauling WorkType's first
   WorkGiver — see docs/superpowers/specs/2026-05-27-auto-haul-design.md."
  [world pawn]
  (let [pos   (:pos pawn)
        cells (zone/stockpile-cells world)]
    (when (seq cells)
      (let [loose (->> (entity/items world)
                       (filter #(and (:pos %)
                                     (not (zone/cell-zoned? world (:pos %)))
                                     (reservation/can-reserve? world (:id %) (:id pawn)))))]
        (when (seq loose)
          (let [item (first (sort-by (juxt #(manhattan pos (:pos %)) :id) loose))
                dest (first (sort-by (juxt #(manhattan (:pos item) %) identity) cells))]
            (job/haul (:id item) dest)))))))
```
